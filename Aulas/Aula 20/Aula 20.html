Aula 20 - Formulários

Nesta aula, aprendi a criar formulários que pode ser utilizados para ler e armazenar informações importantes do usuário.
Para isso, utiliza-se a tag < form > da seguinte maneira:
< form >

Login:< br >
< input type="text" name="Login" >< br >

Senha:< br >
< input type="password" name="Senha" >

< input type="button" value="Entrar" >

< input type="submit" value="Entrar" >
< /form >

Onde type representa o tipo de dados que o formulário deve receber do usuário.
name representa o nome da variável aonde esses dados serão armazenados.
button cria um botão simples.
submit cria um botão que envia o formulário para processamento.
Por fim, value define o nome para o botão.
