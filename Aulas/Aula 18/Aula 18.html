Aula 18 - Tabelas

Nesta aula, aprendi a criar tabelas usando a tag < table > para definir que estamos criando uma tabela, a tag < tr > (Table Row) para criar as linhas e colunas e < td > (Table data) para inserir os dados.
Caso desejado separar as linhas e colunas em células, basta usar o modificador border junto de < table >.
